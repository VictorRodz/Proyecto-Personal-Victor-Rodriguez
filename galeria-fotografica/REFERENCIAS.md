# Referencias

Este archivo documenta todas las fuentes consultadas para aprender Vue 3 por cuenta propia, así como la procedencia y licencia de cada medio utilizado en el proyecto.

## Aprendizaje del framework (Vue 3)

| Recurso | Enlace | Para qué se usó |
|---|---|---|
| Documentación oficial de Vue 3 | https://vuejs.org/guide/introduction.html | Entender Options API, `data`, `computed`, `methods`, `props`, `emits` |
| Guía de componentes de Vue | https://vuejs.org/guide/essentials/component-basics.html | Comunicación entre componentes padre-hijo |
| Guía de Template Refs | https://vuejs.org/guide/essentials/template-refs.html | Acceso al elemento `<audio>` del DOM |
